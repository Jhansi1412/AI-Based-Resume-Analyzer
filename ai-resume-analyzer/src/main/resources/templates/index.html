How it Works

  1. You upload resume & JD (PDF/DOC/DOCX/TXT).
  2. Java extracts text using Apache Tika.
  3. We call OpenAI’s model with both texts.
  4. You get a visual score, skills match, and suggestions.

Tech: Java, Spring Boot, HTML, CSS, Apache Tika, OpenAI API.
